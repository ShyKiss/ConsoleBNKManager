# Console BNK Manager
Allows you to view .bnk file info and replace .wem files by ID 

# Usage
### Drag and drop the `.bnk` file to view the info.
### Drag and drop the desired `.bnk` and `.wem` files (or drop them into the `wems` folder) to replace them.
